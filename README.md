## 🔐 การติดตั้งและตั้งค่า (Setup)

1. ติดตั้งแพ็กเกจ
   ```bash
   npm install
   ```
2. สร้างไฟล์ `.env` (ดูตัวอย่างจาก `.env.example`) แล้วใส่ค่า
   - `CLOUD_NAME`, `API_KEY`, `API_SECRET` — จาก Cloudinary
   - `APP_USERNAME` — ชื่อผู้ใช้สำหรับล็อกอิน
   - `APP_PASSWORD_HASH` — สร้างจากรหัสผ่านด้วยคำสั่ง
     ```bash
     node scripts/hash-password.js "รหัสผ่านของคุณ"
     ```
     แล้วคัดลอกค่าที่ได้มาใส่
   - `SESSION_SECRET` — ข้อความสุ่มยาว ๆ (สร้างได้ด้วย `node -e "console.log(require('crypto').randomBytes(48).toString('hex'))"`)
3. รันเซิร์ฟเวอร์
   ```bash
   npm start
   ```
   แล้วเปิด http://localhost:3000 (ต้องล็อกอินก่อนถึงจะเข้าดูแกลเลอรีได้)

> 🔒 บน **Render** ให้ตั้งค่า env เดียวกันนี้ในหน้า Environment และ **ใช้รหัสผ่านใหม่** (อย่าใช้รหัสเดิมที่เคยฝังในโค้ด)
